## Marketplace (static)

- [`marketplace.html`](marketplace.html) — minimal catalog / install / list / uninstall UI against the API.
- Run the API (`npm run dev`), then open the file in a browser (e.g. `file:///.../apps/web/marketplace.html`) or serve the folder with any static server. Set **API base URL** to match the API (CORS: open from same host as the API if you hit browser restrictions, or add CORS later).
